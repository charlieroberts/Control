      Info

      Control v1.4 is an application enabling configurable control surfaces for mobile devices. For more information, interfaces, tutorials and references please visit the Control website (http://www.charlie-roberts.com/Control).

      Control was created and is maintained by Charlie Roberts, with additional contributions by Greg Shear and Matt Wright. Control was partially realized as a research project for the AlloSphere Research Group.

      Control is open source and was built using the following open source projects (thank you!):

      • PhoneGap www.phonegap.com
      • jQuery Mobile jquerymobile.com
      • oscpack audiomulch.com/~rossb/code/oscpack/
      • pkmFFT github.com/pkmital/pkmFFT
